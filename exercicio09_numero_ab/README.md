# Exercício – Inversão de Valores

Neste exercício, o programa lê dois números (A e B) informados pelo usuário e exibe esses valores em ordem inversa (B, A).

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`int` ou `float`)
- Uso de variáveis
- Atribuição de valores
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa dois números (A e B)
2. O programa armazena os valores em variáveis
3. Os valores são exibidos em ordem inversa, mostrando primeiro B e depois A
