# Exercício – Conversão de Temperatura para Escala Celsius

Neste exercício, o programa lê um valor de temperatura informado em graus Fahrenheit e calcula o valor correspondente em graus Celsius utilizando a fórmula: C = (F − 32) × 5 / 9.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (subtração, multiplicação e divisão)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa a temperatura em graus Fahrenheit
2. O programa aplica a fórmula de conversão para Celsius
3. O valor convertido em Celsius é exibido na tela
