# Exercício – Soma e Diferença de Números

Neste exercício, o programa lê três números informados pelo usuário, calcula a soma dos dois primeiros e a diferença entre os dois últimos.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`int` ou `float`)
- Operações matemáticas básicas (soma e subtração)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa três números
2. O programa calcula a soma do primeiro com o segundo número
3. O programa calcula a diferença entre o segundo e o terceiro número
4. Os resultados são exibidos na tela
