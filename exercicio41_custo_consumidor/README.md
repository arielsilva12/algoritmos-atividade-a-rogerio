# Exercício – Custo ao Consumidor de um Carro

Neste exercício, o programa lê o custo de fábrica de um carro e calcula o custo final ao consumidor, considerando a porcentagem do distribuidor e os impostos aplicados sobre o custo de fábrica.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas
- Uso de variáveis
- Cálculo de porcentagem
- Saída de dados com `print`
- Formatação de valores monetários

## Lógica do Exercício
1. O usuário informa o custo de fábrica do carro  
2. O programa calcula 28% do custo de fábrica referente ao distribuidor  
3. O programa calcula 45% do custo de fábrica referente aos impostos  
4. Os valores do distribuidor, dos impostos e do custo de fábrica são somados  
5. O custo final ao consumidor é exibido com duas casas decimais
