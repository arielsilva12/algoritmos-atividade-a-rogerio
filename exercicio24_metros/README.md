# Exercício – Conversão de Metros para Centímetros

Neste exercício, o programa lê um valor informado em metros (m) e calcula o valor correspondente em centímetros (cm).

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (multiplicação)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa um valor em metros
2. O programa multiplica o valor por 100 para converter para centímetros
3. O valor convertido em centímetros é exibido na tela
