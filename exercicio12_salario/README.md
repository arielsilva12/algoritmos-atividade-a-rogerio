# Exercício – Aumento Salarial de 25%

Neste exercício, o programa lê o salário de um trabalhador informado pelo usuário e calcula o novo salário aplicando um aumento de 25%.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (multiplicação e soma)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa o valor do salário
2. O programa calcula 25% do salário
3. O programa soma o valor do aumento ao salário original
4. O novo salário é exibido na tela
