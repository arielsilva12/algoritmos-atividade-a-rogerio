# Exercício – Conversão de Quilômetros para Metros

Neste exercício, o programa lê um valor informado em quilômetros (km) e calcula o valor correspondente em metros (m).

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (multiplicação)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa um valor em quilômetros
2. O programa multiplica o valor por 1000 para converter para metros
3. O valor convertido em metros é exibido na tela
