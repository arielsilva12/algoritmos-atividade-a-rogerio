# Exercício – Sistema de Equações Lineares (2x2)

Neste exercício, o programa lê os coeficientes de um sistema de equações lineares do tipo  
`ax + by = c` e `dx + ey = f` e calcula os valores de `x` e `y` utilizando as fórmulas matemáticas correspondentes.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`int` e `float`)
- Operações matemáticas
- Uso de parênteses para controle da ordem das operações
- Uso de variáveis
- Saída de dados com `print`
- Expressões algébricas

## Lógica do Exercício
1. O usuário informa os coeficientes `a`, `b` e `c` da primeira equação
2. O usuário informa os coeficientes `d`, `e` e `f` da segunda equação
3. O programa calcula o numerador e o denominador da fórmula de `x`
4. O programa divide o numerador pelo denominador para obter o valor de `x`
5. O programa repete o processo para calcular o valor de `y`
6. Os valores finais de `x` e `y` são exibidos na tela
