# Exercício – Conversão de Velocidade (km/h para m/s)

Neste exercício, o programa lê uma velocidade informada em quilômetros por hora (km/h) e calcula o valor equivalente em metros por segundo (m/s), utilizando a fórmula:

Vm/s = Vkm/h / 3.6

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (divisão)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa uma velocidade em km/h
2. O programa converte esse valor para m/s dividindo por 3.6
3. O resultado da conversão é exibido na tela em metros por segundo
