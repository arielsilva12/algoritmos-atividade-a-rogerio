# Exercício – Cálculo de 70% de um Valor

Neste exercício, o programa lê um valor em real (R$) informado pelo usuário e calcula 70% desse valor.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (multiplicação)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa um valor em reais
2. O programa calcula 70% do valor informado
3. O resultado é exibido na tela
