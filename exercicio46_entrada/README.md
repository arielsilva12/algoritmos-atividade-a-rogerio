# Exercício – Sistema de Pagamento com Entrada e Prestações

Neste exercício, o programa lê o valor de uma mercadoria e calcula o valor da entrada e de duas prestações iguais, inteiras e as maiores possíveis, de acordo com as regras do sistema de pagamento adotado pela loja.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`int`)
- Divisão inteira (`//`)
- Operador resto (`%`)
- Uso de variáveis
- Saída de dados com `print`

## Lógica do Exercício
1. O usuário informa o valor total da mercadoria
2. O programa divide o valor por três utilizando divisão inteira para obter o valor das prestações
3. O resto da divisão é calculado
4. O valor da entrada é obtido somando o valor da prestação ao resto da divisão
5. O programa exibe o valor da entrada e o valor das duas prestações iguais
