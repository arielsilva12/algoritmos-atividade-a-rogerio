# Exercício – Conversão de Meses para Anos e Meses Restantes

Neste exercício, o programa lê um valor inteiro informado em meses e calcula quantos anos e quantos meses restantes esse valor representa.

## Conceitos Utilizados

- Entrada de dados com `input()`
- Conversão de tipos (`int`)
- Operações matemáticas básicas (divisão inteira e resto da divisão)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício

1. O usuário informa um valor inteiro representando a quantidade total de meses.
2. O programa calcula a quantidade de anos dividindo os meses por 12.
3. O programa calcula os meses restantes usando o resto da divisão dos meses por 12.
4. O valor convertido em anos e meses restantes é exibido na tela.
