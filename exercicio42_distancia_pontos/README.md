# Exercício – Distância Entre Dois Pontos no Plano

Neste exercício, o programa lê as coordenadas de dois pontos quaisquer no plano cartesiano e calcula a distância entre eles utilizando a fórmula da distância entre dois pontos.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`int`)
- Operações matemáticas
- Potenciação e raiz quadrada
- Uso de variáveis
- Saída de dados com `print`
- Plano cartesiano

## Lógica do Exercício
1. O usuário informa as coordenadas do primeiro ponto `(x1, y1)`
2. O usuário informa as coordenadas do segundo ponto `(x2, y2)`
3. O programa calcula a diferença entre as coordenadas dos pontos
4. Cada diferença é elevada ao quadrado
5. Os valores ao quadrado são somados
6. O programa calcula a raiz quadrada da soma
7. A distância final entre os dois pontos é exibida na tela
