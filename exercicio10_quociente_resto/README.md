# Exercício – Quociente e Resto da Divisão

Neste exercício, o programa lê dois números inteiros informados pelo usuário e calcula o quociente e o resto da divisão do primeiro número pelo segundo.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`int`)
- Operadores matemáticos:
  - Divisão inteira (`//`)
  - Módulo / resto da divisão (`%`)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa dois números inteiros
2. O programa calcula o quociente da divisão do primeiro número pelo segundo
3. O programa calcula o resto dessa divisão
4. O quociente e o resto são exibidos na tela
