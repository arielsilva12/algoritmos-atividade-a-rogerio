# Exercício – Conversão de Dias para Semanas e Dias Restantes

Neste exercício, o programa lê um valor inteiro informado em dias e calcula quantas semanas completas e quantos dias restantes esse valor representa.

## Conceitos Utilizados

- Entrada de dados com `input()`
- Conversão de tipos (`int`)
- Operações matemáticas básicas (divisão inteira e resto da divisão)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício

1. O usuário informa um valor inteiro representando a quantidade total de dias.
2. O programa calcula a quantidade de semanas usando a divisão inteira de 7.
3. O programa calcula os dias restantes usando o resto da divisão de 7.
4. O valor convertido em semanas e dias restantes é exibido na tela.
