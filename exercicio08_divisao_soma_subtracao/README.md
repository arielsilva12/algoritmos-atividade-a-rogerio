# Exercício – Divisão da Soma pela Subtração

Neste exercício, o programa lê dois números informados pelo usuário e calcula a divisão da soma desses números pela subtração entre eles.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`int` ou `float`)
- Operações matemáticas básicas (soma, subtração e divisão)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa dois números
2. O programa calcula a soma dos dois números
3. O programa calcula a subtração entre o primeiro e o segundo número
4. O programa divide o resultado da soma pelo resultado da subtração
5. O resultado final é exibido na tela
