# Exercício – Conversão de Metros para Quilômetros e Metros

Neste exercício, o programa lê um valor inteiro informado em metros e calcula quantos quilômetros completos e quantos metros restantes esse valor representa.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`int`)
- Operações matemáticas básicas (divisão inteira e resto da divisão)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa um valor inteiro em metros
2. O programa calcula a quantidade de quilômetros usando divisão inteira
3. O programa calcula os metros restantes usando o resto da divisão
4. O valor convertido em quilômetros e metros é exibido na tela
