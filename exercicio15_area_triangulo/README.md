# Exercício – Cálculo da Área de um Triângulo

Neste exercício, o programa lê o valor da base e da altura de um triângulo informado pelo usuário e calcula a sua área usando a fórmula: área = (base * altura) / 2.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (multiplicação e divisão)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa o valor da base do triângulo
2. O usuário informa o valor da altura do triângulo
3. O programa calcula a área utilizando a fórmula: área = (base * altura) / 2
4. O resultado da área é exibido na tela
