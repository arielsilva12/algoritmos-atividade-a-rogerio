# Exercício – Média Ponderada de 3 Notas

Neste exercício, o programa lê três notas de um aluno e o peso de cada nota informado pelo usuário, e calcula a média ponderada.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (multiplicação, soma e divisão)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa três notas do aluno
2. O usuário informa o peso de cada nota
3. O programa calcula a soma dos produtos de cada nota pelo seu respectivo peso
4. O programa divide essa soma pelo total de pesos para obter a média ponderada
5. A média ponderada é exibida na tela
