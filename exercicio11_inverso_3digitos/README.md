# Exercício – Inversão de Número de 3 Dígitos

Neste exercício, o programa lê um número inteiro de 3 dígitos e calcula o número inverso, reorganizando a centena, dezena e unidade.  
Exemplo: número = 532 → inverso = 235.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`int`)
- Operadores matemáticos:
  - Divisão inteira (`//`)
  - Módulo / resto da divisão (`%`)
- Uso de variáveis
- Operações matemáticas básicas
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa um número inteiro de 3 dígitos
2. O programa separa a centena, a dezena e a unidade
3. O número inverso é formado reorganizando os dígitos (unidade, dezena e centena)
4. O número invertido é exibido na tela
