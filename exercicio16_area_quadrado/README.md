# Exercício – Cálculo da Área de um Quadrado

Neste exercício, o programa lê o valor do lado de um quadrado informado pelo usuário e calcula a sua área usando a fórmula: área = lado².

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (exponenciação)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa o valor do lado do quadrado
2. O programa calcula a área utilizando a fórmula: área = lado²
3. O resultado da área é exibido na tela
