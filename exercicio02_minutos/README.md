# Exercício 02 – Conversão de Horas para Minutos

Este projeto é um exercício simples em Python que lê um valor em horas e um valor em minutos, calcula o total de minutos e exibe o resultado.

O objetivo do exercício é praticar:
- Entrada de dados
- Operações matemáticas básicas
- Uso de variáveis
- Saída de dados no terminal

---

# Lógica do Problema

1. Ler um valor inteiro representando horas  
2. Ler um valor inteiro representando minutos  
3. Converter horas para minutos (1 hora = 60 minutos)  
4. Somar os minutos obtidos  
5. Exibir o total de minutos
