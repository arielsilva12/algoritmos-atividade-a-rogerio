# Exercício – Composição do Latão (Cobre e Zinco)

Neste exercício, o programa lê a quantidade total de latão (em quilogramas) informada pelo usuário e calcula a quantidade de cobre e de zinco presentes, sabendo que o latão é composto por 70% de cobre e 30% de zinco.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas
- Cálculo de porcentagem
- Uso de variáveis
- Saída de dados com `print`

## Lógica do Exercício
1. O usuário informa a quantidade total de latão em quilogramas
2. O programa calcula 70% desse valor para obter a quantidade de cobre
3. O programa calcula 30% desse valor para obter a quantidade de zinco
4. Os valores correspondentes a cada componente são exibidos na tela
